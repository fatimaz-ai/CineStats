# CineStats 🎬

A movie taste analyzer built with Python, Flask, and the TMDB API. Enter movies you love, and it analyzes your genre patterns, gives you a "movie personality," and recommends similar films.

**Live demo:** https://tinyurl.com/fz-cinestats

## Features
- Genre analysis across your movie list using the TMDB API
- Personality result with a matching quote
- Interactive pie chart breakdown of your genre taste
- Movie recommendations based on your list
- A public guestbook for comments

## Built with
- Python, Flask
- TMDB API
- Chart.js
- Vanilla HTML/CSS/JS frontend

## A note on how this was built
I built the backend logic myself: API integration, genre analysis, personality/recommendation logic, and the comment system. For the frontend design, I worked closely with AI assistance to shape the visual direction. I spent the weeks before this project learning Python fundamentals and understand how the code works.
